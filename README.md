# My-CTF-Journey

A collection of my solved Capture The Flag challenges, including write-ups, solutions, and custom tooling.

This repository documents my progress across different platforms such as TryHackMe, focusing on learning, problem-solving, and practical cybersecurity skills.

All challenges are initially worked on in a separate private repository. This public repository only contains the finalized write-ups and selected solution scripts.


## Structure

Each challenge is organised by platform and, if suitable, by event or category.

Example:

RootMe/
- `readme.md` → explanation of the challenge and solution  
- `solve.py` → script used to solve the challenge  

## Scripts

This repository also contains custom bash scripts used in the CTFs. They are mainly used to improve the workflow  and quality of life.

The scripts are continuously refined and adapted over time.

## Goals

- Document solved challenges
- Improve problem-solving skills
- Build reusable tooling
- Track long-term progress in cybersecurity


## Disclaimer

All challenges belong to their respective platforms (e.g. TryHackMe).

This repository only contains my personal solutions and write-ups.

## Notes

- Write-ups may vary in detail depending on the challenge
- Some solutions can focus on the key idea rather than full step-by-step explanations
